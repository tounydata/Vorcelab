# Liste de courses GymVisual — 75 démos d'exercices (~75 $)

> **Mode d'emploi** (~30-45 min) :
> 1. Crée un compte sur **https://gymvisual.com** (email suffit).
> 2. Pour chaque ligne : colle le **terme de recherche** dans la barre de
>    recherche du site → choisis la version **GIF animé** (pas l'illustration
>    statique) qui correspond le mieux au libellé Vorcelab → ajoute au panier.
>    ⚠️ Reste sur le **même style/personnage** (3D masculin ou féminin, mais le
>    même partout) pour un rendu cohérent dans l'app.
> 3. S'il y a plusieurs variantes proches, prends la plus simple (pas de machine
>    exotique). Si un exercice est introuvable, prends la variante la plus
>    proche et note-le — on ajustera le libellé dans l'app si besoin.
> 4. Paye (les GIF passent à ~0,90 $/pièce au-delà de 10 articles), télécharge
>    tout, fais un ZIP et envoie-le-moi dans le chat. Je m'occupe de tout le
>    reste (conversion WebP allégée, ré-hébergement, mapping, web + mobile,
>    nettoyage des sources en zone grise).
